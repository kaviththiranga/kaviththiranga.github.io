Methods

getNextClient

(Client[] loadBalanceCallerActionsArray)

Provides an HTTP client which is choosen according to the algorithm.

Parameters

  • loadBalanceCallerActionsArray Client[]

  • Array of HTTP clients which needs to be load balanced

  • Return Type

    (Client | error)

  • Choosen Client from the algorithm or an error for the failure in the algorithm implementation
